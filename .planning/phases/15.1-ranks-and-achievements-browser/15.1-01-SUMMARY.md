# Phase 15.1 Plan 01: RanksAndAchievementsViewModel ‚Äî Summary

Shared presentation layer for the rank ladder screen. Combines `GamificationRepository.rankState` + `GamificationRepository.totalXp` into a `RankLadderUiState` containing 10 `RankRow`s classified as PASSED / CURRENT / LOCKED per the D-151-07 derivation rules, annotated with `@NativeCoroutinesState` for iOS consumption.

## Derivation Rules (D-151-07)

| RankState | Row classification |
|-----------|-------------------|
| `Unranked` | All 10 rows LOCKED. SILVER gets `xpToReach = 0` (D-11 first-workout hint). Other rows get `xpToReach = threshold`. |
| `Ranked(currentRank)` | Rows below `currentRank.ordinal` = PASSED (`xpToReach = null`). Row at ordinal = CURRENT (`xpToReach = null`). Rows above = LOCKED with `xpToReach = max(0, threshold - totalXp)`. |

Rows always ordered by `Rank.values()` ‚Äî SILVER (ordinal 0) ‚Üí GLOBAL_ELITE (ordinal 9).
