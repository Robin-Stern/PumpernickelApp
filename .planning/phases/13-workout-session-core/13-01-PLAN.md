```kotlin
fun DrumPicker(
    items: List<Int>,
    selectedItem: Int,
    onItemSelected: (Int) -> Unit,

    visibleItemCount: Int = 5

```

**Implementation details:**

1. **LazyListState + SnapFlingBehavior:**
   - `val listState = rememberLazyListState()`
   - Use `rememberSnapFlingBehavior(lazyListState = listState)` from `androidx.compose.foundation.gestures.snapping` (requires `@OptIn(ExperimentalFoundationApi::class)`)
   - LazyColumn with `flingBehavior = snapFlingBehavior`

2. **Item layout:**
   - Each item is a fixed-height Box (itemHeight = total picker height / visibleItemCount)
   - Total picker height: `visibleItemCount * itemHeight` (use 200.dp total, so itemHeight = 40.dp for 5 visible items)
   - Add empty spacer items at top and bottom (visibleItemCount / 2 items) so the first and last real items can scroll to center

3. **Visual highlighting (Claude's discretion for styling):**
   - Center item: `MaterialTheme.typography.headlineMedium`, full alpha (1f), `FontWeight.Bold`
   - +/-1 items: `MaterialTheme.typography.titleMedium`, alpha 0.5f
   - +/-2 items: `MaterialTheme.typography.bodyMedium`, alpha 0.25f
   - Calculate distance from center using `listState.firstVisibleItemIndex` and `listState.firstVisibleItemScrollOffset`
   - Apply alpha based on distance: `alpha = maxOf(0.2f, 1f - (distance * 0.3f))`

4. **Center selection indicator:**
   - Draw horizontal divider lines above and below center item position using `Divider` composable (or `HorizontalDivider`)
   - Use `MaterialTheme.colorScheme.primary` with 0.3f alpha for divider color

5. **Selection sync:**
   - On composition, scroll to the initial `selectedItem` index: `LaunchedEffect(selectedItem) { listState.scrollToItem(index) }` where index accounts for top spacer items
   - Detect settled position using `LaunchedEffect(listState.isScrollInProgress)`: when scrolling stops, read the centered item index from `listState.firstVisibleItemIndex + visibleItemCount / 2` and call `onItemSelected(items[centeredIndex])`
   - Use `snapshotFlow { listState.isScrollInProgress }` to reliably detect scroll end:
     ```kotlin
     LaunchedEffect(listState) {
         snapshotFlow { listState.isScrollInProgress }
             .filter { !it }
             .collect {
                 val centerIndex = listState.firstVisibleItemIndex + visibleItemCount / 2
                 val actualIndex = centerIndex // spacer items already accounted for via padding
                 if (actualIndex in items.indices) {
                     onItemSelected(items[actualIndex])
                 }
             }
     }
     ```

6. **Label above picker:**
   - If `label` is not empty, show `Text(label)` above the LazyColumn with `MaterialTheme.typography.labelMedium` and `color = MaterialTheme.colorScheme.onSurfaceVariant`

7. **Display transform:**
   - Each item's text uses `displayTransform(item)` so weight values can be formatted as "50.0 kg" while storing kgX10 = 500
